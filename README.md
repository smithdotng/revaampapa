# RevaampAPA - Aggregation, Promotions & Agency Platform

![RevaampAPA Logo](/public/img/logo.png)

RevaampAPA is Nigeria's leading aggregation, promotions and agency platform for property and project management.

## 🚀 Features

### For Property Owners
- List unlimited properties with ₦20,000 verification fee
- Get 10% commission on sales
- Reach thousands of potential buyers
- Payment verification system

### For Property Promoters (FREE Registration)
- **Main Benefits:**
  - Introduce Property: 40% of Agency Fee
  - Introduce Buyer: 90% from buyer + 50% from seller
  - Introduce Promoter: 70% residual income
  
- **Additional Benefits:**
  - Hotel Onboarding: 30% commission
  - Room Voucher Subscriber: 50% subscription fee
  - Business Partner Referral: ₦10,000
  - Sub-Aggregator Referral: ₦25,000

### For Business Partners (₦20,000)
- White-label solutions
- Dedicated account manager
- Marketing materials & training
- Priority technical support

### For Project Management Subscribers
- Full project management suite
- Real-time progress tracking
- Document management system
- Client portal access

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Frontend:** EJS templates, Plax Theme
- **Payment:** Paystack Integration
- **Authentication:** Express Session
- **Email:** Nodemailer

## 📋 Prerequisites

- Node.js (v18+)
- MongoDB
- Paystack account (for payments)
- Gmail account (for email notifications)
